# CSS3 Features Demo

This project is a simple demonstration of some important **CSS3 features**.  
It is created for practice and learning purposes.

## Features Used

The project includes examples of:

- **CSS Transform**
  - rotate()
  - scale()
  - translate()
- **CSS Transition**
  - smooth effect when hovering over elements
- **CSS Overflow**
  - handling content that goes outside the box

## Project Purpose

The main purpose of this project is to understand how CSS3 features work in a practical way.  
It helps beginners learn how to style elements, add motion, and control content layout.

## Technologies Used

- HTML5
- CSS3

## How to Run the Project

1. Download or extract the project folder
2. Open the folder
3. Double-click the `index.html` file  
   or  
   Open `index.html` in any web browser

## Learning Outcomes

By working on this project, you can learn:

- how to apply CSS3 transform properties
- how to use transition for interactive effects
- how overflow works in containers
- how CSS can improve the look and behavior of a webpage
